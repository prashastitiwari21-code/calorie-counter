# Calorie Counter 

A simple web-based calorie tracking form that helps users monitor their daily calorie intake and determine whether they are in a calorie deficit or surplus.

---

# What It Does

Users enter their daily calorie budget along with calorie counts for different meals. The app calculates the total consumed calories and compares it against the budget to show whether the user is under or over their goal.

---

## Features

- Set a daily calorie budget
- Log calories for multiple meals (breakfast, lunch, dinner, snacks, etc.)
- Calculates total consumed calories dynamically
- Displays calorie deficit or surplus after submission
- Clear/reset functionality to start a new entry

---

## Tech Stack

- **HTML** — form structure and inputs
- **CSS** — basic styling
- JavaScript (Vanilla) — DOM manipulation, input handling, and calorie calculation logic

---

## How to Use

1. Clone or download this repository
2. Open `index.html` in your browser — no installation needed
3. Enter your daily calorie budget
4. Add calorie values for each meal
5. Click **Calculate** to see your deficit or surplus
6. Click **Clear** to reset the form
